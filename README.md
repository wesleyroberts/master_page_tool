# Master Page Tool

Este modulo contem um portlet JSP para administracao em massa de Master Pages no Liferay.

O objetivo do portlet e permitir que o usuario:

- visualize os Master Pages disponiveis no site
- selecione varias paginas por meio de checkboxes
- aplique um Master Page em lote para as paginas escolhidas

## Ambiente utilizado

O setup deste projeto foi preparado para o portal:

- `portal-7.4-ga132`

## Compatibilidade

Esta branch foi pensada para ambientes baseados em Liferay 2025.
